图标字形（icon font）

什么是图标字形（icon font）

图标是图形界面中必不可少的设计元素。在传统网页中，图标往往使用图片格式，以img标签或者css background的形式在页面中渲染出来。

使用img标签显示一个图标：

<img src="assets/04/icon_rocket.png" alt="">

使用css background显示一个图标：

.rocket-bg{ background-image: url(icon_rocket.png); background-size: contain; width:50px; height:50px; }
上面的方法虽然可以把图标显示出来，但也有一些显著的缺点：
  • 如果图标是位图格式（通常是png），那么放大页面时或者使用高清屏浏览时，图标会变模糊；
  • 理论上你也可以使用web矢量格式svg作为图片和背景插入页面，svg虽然能保证清晰度，但浏览器兼容性不好；
  • 很多时候，UI上的图标会有多个状态效果用于响应用户的鼠标悬浮、点击等行为，如果使用img或者css background的方式来显示这些图标，就要为每个状态效果输出一个图标文件版本，管理和维护的工作量会翻倍。

为了在网页上显示清晰锐利的图标，同时又确保浏览器兼容性，开发人员使用工具将所有图标都打包为一个字体文件，让页面中的图标以字形（而不是位图）方式渲染，这种字形化的图标就是icon font。
